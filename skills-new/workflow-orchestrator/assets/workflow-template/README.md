# Workflow 目录说明

本目录用于完整开发 workflow 的单需求执行。

## 文件职责

- `source-index.md`：需求资料、原型、PRD、接口草案索引
- `requirements.md`：需求评审、范围、业务域、AC
- `requirements-gate.md`：需求门禁记录
- `tech-research.md`：技术最新信息、官方文档、技术选型证据
- `solution.md`：前后端技术方案、接口和数据模型映射
- `solution-gate.md`：技术方案门禁记录
- `frontend-plan.md`：前端实施计划
- `backend-plan.md`：后端实施计划
- `plan-gate.md`：实施计划门禁记录
- `task-ledger.md`：执行任务状态总账
- `verification.md`：测试与验收映射
- `handoff.md`：实施计划验收汇报与残余风险
- `sql/`：SQL 草案
