## 使用说明

- **spec_id**: 规范唯一标识
- **file**: 规范文件路径（相对 .docs/spec/）
- **stages**: 适用阶段（understand, design, build, done）
- **projects**: 适用项目（all 或具体项目名）
- **tags**: 标签
- **status**: 状态（active, deprecated）
- **description**: 规范描述
